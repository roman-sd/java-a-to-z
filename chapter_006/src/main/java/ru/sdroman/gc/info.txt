2. Размер объекта User (64 битная)

- заголовок для user 12 байт;
- ссылочная переменная на name 4 байта
- num 4 байта
- выравнивание 4 байта

- заголовок для поля name 12 байт + 4 байта на длину массива = 16 байт;
- поля int 3 * 4 байт = 12 байт; (offset, count, hash)
- выравнивание 4 байта;
- char 2 байта * 4  = 8 байт;
итого: 66 байт.

3. пустой объект занимает 8 байт для 32 битная система
                         16 байт     64 битная система
