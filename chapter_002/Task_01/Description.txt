1. Класс Item описывает модель заявки. Включает поля: 
 - имя(name) 
 - описания(description) 
 - id 
 - комментарий (comments)
а так же методы:
 - полученя и установки имени
 - полученя и установки описания
 - полученя и установки id
 - полученя и установки комментария

2. Класс Tracker используется для хранения заявок. Содержит поля:
 - массив заявок(items)
 - позиция для работы с массивом заявок(position)
 - константа RN для генерации id
 методы:
 - добавление заявки(add)
 - поиск по id(findById)
 - поиск по имени(findByName)
 - генератор id (generateId)
 - получение всех заявок(getAll)
 - редактирование заявок(edit)
 - удаление заявок(remove)
 - добавление комментария(addComment)

3. Класс StartUI используется для запуска программы, содержит метод main

4. Класс Menu взаимодействут с пользователем и управляет хранилищем заявок.
